public modifyWifiShop($data)
说明
修改WiFi门店信息
参数说明
类型 名称 说明 备注
string $data.shop_id 门店ID(shop_id)
string $data.old_ssid 原ssid
string $data.ssid 新ssid 32个字符以内;ssid支持中文,但可能因设备兼容性问题导致显示乱码,或无法连接等问题, 当门店下是portal型设备时,ssid必填;当门店下是密码型设备时,ssid选填,且ssid和密码必须有一个以大写字母"WX"开头
string $data.password 无线网络设备的密码 8-24个字符;不能包含中文字符;当门店下是密码型设备时,才可填写password,且ssid和密码必须有一个以大写字母"WX"开头
调用示例 
$data = [
	"shop_id" => 429620,
    "old_ssid" => "WX123",
    "ssid" => "WX567"
];
Wechat::getInstance()->modifyWifiShop($data);
返回参数说明
boolean
返回参数示例 
true
