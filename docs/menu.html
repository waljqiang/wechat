public function setMenu($options)
说明
创建微信公众号菜单
参数说明
类型 名称 说明 备注
array button 一级菜单数组 个数为1-3个
array sub_button 二级菜单数组 非必须,个数为1-5个
string type 菜单类型 view,网页类型;click,点击类型;miniprogram,小程序类型
sting name 菜单标题 一级菜单不超过16个字节,子菜单不超过60个字节
string key 菜单KEY值 click等点击类型必须,用于消息接口推送,不超过128字节
string url 网页链接,用户点击菜单可打开链接 view、miniprogram类型必须,不超过1024字节
string media_id 新增永久素材接口返回的合法media_id media_id类型和view_limited类型必须
string appid 小程序的appid miniprogram类型必须
string pagepath 小程序的appid 小程序的页面路径
调用示例 
示例1:
$wechat->setMenu([
	"button" => [
		[
			"type" => "view",
			"name" => "我要下单",
			"url" => "http://www.baidu.com"
		],
		[
			"type" => "click",
			"name" => "个人中心",
			"key" => "V0001_PERSONAL"
		],
		[
			"type" => "click",
			"name" => "关于我们",
			"key" => "V0002_ABOUT"
		]
	]
]);
	                    
示例2:
$menu = new Menu($wechat);
$menu->setMenu([
	"button" => [
		[
			"type" => "view",
			"name" => "我要下单",
			"url" => "http://www.baidu.com"
		],
		[
			"type" => "click",
			"name" => "个人中心",
			"key" => "V0001_PERSONAL"
		],
		[
			"type" => "click",
			"name" => "关于我们",
			"key" => "V0002_ABOUT"
		]
	]
]);
返回参数说明
boolean
返回参数示例

public function getMenu()
说明
获取公众号菜单
参数说明
无
调用示例 
示例1:
$wechat->getMenu();
	                    
示例2:
$menu = new Menu($wechat);
$menu->getMenu();
返回参数说明
类型 名称 说明 备注
integer is_menu_open 菜单是否开启 0,未开启;1,开启
array selfmenu_info 菜单信息
array button 菜单按钮
string type 菜单类型 菜单类型有view(跳转网页)、text(返回文本,下同)、img、photo、video、voice.使用API设置的则有8种,详见微信公众号《自定义菜单创建接口》
string name 菜单名称
mixed value/key/url 菜单值 对于不同的菜单类型,value的值意义不同.微信公众号官网上设置的自定义菜单:Text:保存文字到value; Img、voice：保存mediaID到value;Video：保存视频下载链接到value;News:保存图文消息到news_info,同时保存mediaID到value;View：保存链接到url.使用API设置的自定义菜单:click、scancode_push、scancode_waitmsg、pic_sysphoto、pic_photo_or_album、 pic_weixin、location_select：保存值到key;view：保存链接到ur
array news_info 图文消息的信息
string title 图文消息的标题
string digest 摘要
string author 作者
integer show_cover 是否显示封面 0,不显示;1,显示
string cover_url 封面图片的URL
string content_url 正文的URL
string source_url 原文的URL 若置空则无查看原文入口
