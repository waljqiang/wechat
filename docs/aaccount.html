private getTicket($data)
说明
获取二维码ticket
参数说明
类型 名称 说明 备注
integer $data.expire_seconds 二维码有效时间,以秒为单位. 最大不超过2592000（即30天),此字段如果不填,则默认有效期为30秒
string $data.action_name 需要获取的二维码类型 QR_SCENE,临时的整型参数值;QR_STR_SCENE,临时的字符串参数值;QR_LIMIT_SCENE,永久的整型参数值;QR_LIMIT_STR_SCENE,永久的字符串参数值
string $data.action_info 二维码详细信息 QR_SCENE,临时的整型参数值;QR_STR_SCENE,临时的字符串参数值;QR_LIMIT_SCENE,永久的整型参数值;QR_LIMIT_STR_SCENE,永久的字符串参数值
integer $data.action_info.scene.scene_id 场景值ID 当二维码类型对应整形参数时需要此值,临时二维码时为32位非0整型,永久二维码时最大值为100000(目前参数只支持1--100000)
string $data.action_info.scene.scene_str 场景值ID 当二维码类型对应字符串参数类型时需要此值,长度限制为1到64
调用示例 
无
返回参数说明
类型 名称 说明 备注
string ticket 二维码ticket
integer expire_seconds 二维码有效时间
string url 二维码图片解析后的地址 开发者可根据该地址自行生成需要的二维码图片

public getQrcode($str,$type = "QR_SCENE",$expire = 30)
说明
生成带参数的二维码
参数说明
类型 名称 说明 备注
mixed $str 二维码参数,为数字或字符串 具体格式要求请查看微信公众号文档相关说明
string $type 二维码类型 QR_SCENE临时的整型参数值;QR_STR_SCENE,临时的字符串参数值;QR_LIMIT_SCENE,永久的整型参数值;QR_LIMIT_STR_SCENE,永久的字符串参数值,默认为QR_SCENE
integer $expire 二维码过期时间 默认为30秒
调用示例 
示例1:
Wechat::getInstance()->getQrcode(12);
	                    
示例2:
$wechat = Wechat::getInstance();
$account = new Account($wechat);
$account->getQrcode(12);
