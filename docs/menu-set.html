class
Waljqiang\Wechat\Handles\Menu::class
public setMenu($options)
说明
创建微信公众号菜单
参数说明
类型 名称 说明 备注
array button 一级菜单数组 个数为1-3个
array sub_button 二级菜单数组 非必须,个数为1-5个
string type 菜单类型 view,网页类型;click,点击类型;miniprogram,小程序类型
sting name 菜单标题 一级菜单不超过16个字节,子菜单不超过60个字节
string key 菜单KEY值 click等点击类型必须,用于消息接口推送,不超过128字节
string url 网页链接,用户点击菜单可打开链接 view、miniprogram类型必须,不超过1024字节
string media_id 新增永久素材接口返回的合法media_id media_id类型和view_limited类型必须
string appid 小程序的appid miniprogram类型必须
string pagepath 小程序的appid 小程序的页面路径
调用示例 
示例1:
Wechat::getInstance()->setMenu([
"button" => [
	[
		"type" => "view",
		"name" => "我要下单",
		"url" => "http://www.baidu.com"
	],
	[
		"type" => "click",
		"name" => "个人中心",
		"key" => "V0001_PERSONAL"
	],
	[
		"type" => "click",
		"name" => "关于我们",
		"key" => "V0002_ABOUT"
	]
]
]);
