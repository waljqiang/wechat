class
Waljqiang\Wechat\Handles\Reply::class
public replyUser($messageType,$message)
说明
生成发送给公众号的消息
参数说明
类型 名称 说明 备注
string $messageType 消息类型
array $message 消息
调用示例 
$message = [
"ToUserName" => "o9lXF0oPTBOMS44dILU1kfZMlra0",
"FromUserName" => "o9lXF0oPTBOMS44dILU1kfZMlra0",
"Content" => "您好"
];
Wechat::getInstance()->replyUser(Waljqiang\Wechat\Handles::TEXT,$message);
返回参数说明
array xml消息体转变成数组格式消息体
返回参数示例
"<xml><ToUserName><![CDATA[o9lXF0oPTBOMS44dILU1kfZMlra0]]></ToUserName> <FromUserName><![CDATA[o9lXF0oPTBOMS44dILU1kfZMlra0]]></FromUserName> <CreateTime>1583820730</CreateTime> <MsgType><![CDATA[text]]></MsgType> <Content><![CDATA[您好]]></Content> </xml>"
