class
Waljqiang\Wechat\Handles\Message::class
public createKfAccount($data)
说明
创建客服账号
参数说明
类型 名称 说明 备注
string kf_account 客服账号 格式:账号前缀@公众号微信号
string kf_nick 客服昵称 最长6个汉字或12个英文字符
string password 客服密码 格式为密码明文的32位加密MD5值。该密码仅用于在公众平台官网的多客服功能中使用，若不使用多客服功能，则不必设置密码
调用示例 
示例1:
Wechat::getInstance()->creatKfAccount([
"kf_account" => "test1@test",
"nickname" => "客服1",
"password" => md5("123456")
]);
                    
示例2:
$wechat = Wechat::getInstance();
$message = new Message($wechat);
$menu->creatKfAccount([
"kf_account" => "test1@test",
"nickname" => "客服1",
"password" => md5("123456")
]);
返回参数说明
boolean
返回参数示例 
true
