public getUserInfo($openID,$lang="zh_CN")
说明
获取用户基本信息
参数示例
类型 名称 说明 备注
string $openID 用户openid
string $lang 语言 默认为zh_CN,支持参数请查看微信获取用户基本信息接口说明
调用示例 
Wechat::getInstance()->getUserInfo("o6_bmjrPTlm6_2sgVt7hMZOPfL2M");
返回参数说明
类型 名称 说明 备注
integer subscribe 用户是否关注该公众号标识 0,未关注;1:关注
string openid 用户openID
string nickname 用户的昵称
integer sex 用户的性别 值为1时是男性，值为2时是女性，值为0时是未知
string language 用户的语言
string city 用户所在城市
string province 用户所在省份
string country 用户所在国家
string headimgurl 用户头像 最后一个数值代表正方形头像大小（有0、46、64、96、132数值可选，0代表640*640正方形头像），用户没有头像时该项为空。若用户更换头像，原有头像URL将失效
integer subscribe_time 用户关注时间
string unionid 用户标识 只有在用户将公众号绑定到微信开放平台帐号后，才会出现该字段
string remark 用户备注
integer groupid 用户所在的分组ID
array tagid_list 用户被打上的标签ID列表
string subscribe_scene 用户关注的渠道来源 ADD_SCENE_SEARCH 公众号搜索，ADD_SCENE_ACCOUNT_MIGRATION 公众号迁移，ADD_SCENE_PROFILE_CARD 名片分享，ADD_SCENE_QR_CODE 扫描二维码，ADD_SCENE_PROFILE_LINK 图文页内名称点击，ADD_SCENE_PROFILE_ITEM 图文页右上角菜单，ADD_SCENE_PAID 支付后关注，ADD_SCENE_OTHERS 其他
integer qr_scene 二维码扫码场景值
string qr_scene_str 二维码扫码场景值
