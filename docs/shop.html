public function createShop($data)
说明
创建门店
参数说明
类型 名称 说明 备注
string $data.sid 商户自己的id,用于后续审核通过收到poi_id 的通知时，做对应关系。请商户自己保证唯一识别性 非必须
string $data.business_name 门店名称,15个汉字或30个英文字符内,不应包含地区、地址、分店名等信息
string $data.branch_name 分店名称,20个字,不应包含地区信息，不应与门店名有重复
string $data.province 门店所在省,10个字以内
string $data.city 门店所在市,10个字以内
string $data.district 门店所在区,10个字以内
string $data.address 门店所在详细街道地址,不要填写省市信息
string $data.telephone 门店电话,纯数字,区号,分机号均由"-"隔开
string $data.categories 门店分类,不同级分类用","隔开
integer $data.offset_type 坐标类型,坐标类型:1,为火星坐标;2,sogou经纬度;3,为百度经纬度;4,mapbar经纬度;5,GPS坐标;6,sogou墨卡托坐标 注：高德经纬度无需转换可直接使用
float $data.longitude 门店所在经度
float $data.latitude 门店所在维度
array $data.photo_list 门店图片列表 非必须
string $data.recommend 推荐品,不超过200字 非必须
string $data.special 特殊服务,不超过200字
string $data.introduction 商户简介,主要介绍商户信息等 300字以内 非必须
string $data.open_time 营业时间,24 小时制表示,用"-"连接,如 8:00-20:00 非必须
integer $data.avg_price 人均价格 非必须
