class
Waljqiang\Wechat\Handles\Message::class
public getKfAccount()
说明
获取所有客服账号
参数说明
调用示例 
示例1:
Wechat::getInstance()->getKfAccount();
                    
示例2:
$wechat = Wechat::getInstance();
$menu = new Menu($wechat);
$menu->getKfAccount();
返回参数说明
类型 名称 说明 备注
string kf_account 客服账号 格式:账号前缀@公众号微信号
string kf_nick 客服昵称 最长6个汉字或12个英文字符
string kf_id 客服工号
string kf_headimgurl 客服头像
